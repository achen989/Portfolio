Ongoing Projects

I am working on a Java Program that solves a maze using graph theory. The user inputs a maze and the algorithm solves the inputted maze. The algorithm searches every possible path of the maze, and adds each possible path to a list. If a dead end is reached, the path is deleted. If it is successful, the path is illustrated to the user. The program uses an array storing which squares are walls or walkable, in addition to a visited array that keeps track of squares already visited.
